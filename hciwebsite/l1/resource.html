Usability Heuristics

#1 Visibility of System Status

As a designer you need to make sure that the user knows what is going on with the system at the current time and the user should be given some form of appropriate feedback that allows them to understand the current status of the system.

  • Article from the NN group
  • Article from Medium - Harsh Gorasia
#2 Match Between the System and the Real-World

The system's design should be sure to speak the same language as its users. Users should see familiar phrases, icons, images to that of the real-world. This allows a natural, logical and familiar understanding to the aspects of your design for whoever is using it.

  • Article from the NN group
  • Article from Medium - Harsh Gorasia
#3 User Control and Freedom

Sometimes users up somewhere they shouldn't, or don't want to be. Your design needs to give users the feeling of control and that they have the ability to navigate the design at their own will such as fixing mistakes, correcting choices and finding a simple way to return to where they previous were.

  • Article from the NN group
  • Article from Medium - Harsh Gorasia
#4 Consistency and Standards

Consistency provides users with the familiarity, patterns and logic for the user. It is about adhering to the conventions of your design and making sure that your design is consistent in itself as well as aligning itself with all other designs that are similar to yours. Both internal and external consistency contribute to learnability and usability of your designs.

  • Article from the NN group
  • Article from Medium - Harsh Gorasia

Bootstrap

Bootstrap is a free, third-party HTML, CSS and JS library that is built for easy-to-use responsive website development. When using Bootstrap make sure to link the appropriate versions of CSS and JS so that all of the built-in features and animations work. Alternatively, You can download the Bootstrap library and the SASS files to more directly access and edit the library and CSS classes.

The link below takes you to the Bootstrap website that contains the CDN and the documentation for its library.
