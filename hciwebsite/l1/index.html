About

This website was designed and developed as a learning tool for students to study Neilson's 10 usability heuristics. The website itself is used as a template for web development and understanding the usability heuristics.

You will use this website as the basis for understanding how to develop a website. Once the website has been completed you will also be able to use it as a resource base for the Usability Heuristics and making sure that you understand exactly what they are and what some examples are of each heuristic.
